# Security Policy

Do not open a public issue for a suspected vulnerability, exposed credential, unsafe
automation path, or malicious bundled asset.

Report security concerns through a private
[GitHub security advisory](https://github.com/kolega-ai/kolega-skills/security/advisories/new).
Include the affected skill or script, reproduction steps, potential impact, and any proposed
mitigation. Do not include real secrets or sensitive user data.

Security fixes are prioritized according to exploitability and impact. Supported content is
the current `main` branch; users should update to the latest revision before reporting an
issue already fixed there.
